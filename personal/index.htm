PARCEL MANAGEMENT SYSTEM

A logistics application designed to track and manage packages from dispatch to final delivery. This system provides real-time status updates, manages courier routes, and maintains a complete inventory of all parcels.

RESTURANT MANAGEMENT SYSTEM

A comprehensive system to streamline restaurant operations. It handles online ordering, table reservations, and menu management. This project is designed to improve efficiency from the kitchen to the front-of-house.

An all-in-one administrative portal for a university. This application manages student records, course registration, and faculty schedules, simplifying the core academic and administrative workflows of the institution.
